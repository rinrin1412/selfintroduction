自己紹介ファイルの作り方

手順１：1から４のうち適当に自分のファイルを選んでください。数字は早い者勝ち（かぶらないように）

手順2:htmlファイルをメモ帳、テキストエディタなどで開いてください

手順3：文字が書いてある部分を自分用に書き直してください
	　初めの数字を自分の名前に書き直してください（ファイル名はそのまま）

手順４：時間割のスクリーンショットを"自分の番号.jpg"に上書きしてください

手順５：firefoxなどのブラウザで表示できるか確認してください

出来ていればおしまい


注）ファイルの名前は基本的に上書きで、名前を換えないでください
　　トップのファイルは書き換えないでください

質問があればここに書き足してください
